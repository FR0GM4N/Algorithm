## 코드

> 위 그림의 `f()` 대신 `paper_cut()` 이라는 함수명을 사용했다.

```python
def paper_cut(n):
    if n == 1:
        return 1
    elif n == 2:
        return 3
    return paper_cut(n-1) + paper_cut(n-2) * 2


T = int(input())
for tc in range(1, T+1):
    N = int(input())
    ans = paper_cut(N//10)

    print("#{} {}".format(tc, ans))
```
